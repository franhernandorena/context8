# Dev Workflows — Agent Context

Plugin de prompts estructurados para asistentes de codigo AI.

## Quick Start

1. Lee `AGENT_CONTEXT.md` primero — contexto completo del proyecto
2. Lee `AGENT_SYSTEM_PROMPT.md` para el system prompt inicial
3. Revisa `tasks/` para tareas pendientes o en progreso
4. Usa `PROJECT_OVERVIEW.md` para orientacion rapida

## Index

| File | Proposito |
|------|-----------|
| `AGENT_CONTEXT.md` | Contexto completo del proyecto (tech stack, modulos, patrones) |
| `AGENT_SYSTEM_PROMPT.md` | System prompt listo para copiar/pegar |
| `PROJECT_OVERVIEW.md` | Resumen de 1 pagina del proyecto |
| `architecture/data_flow.md` | Flujo de datos del plugin |
| `architecture/key_patterns.md` | Patrones de diseno y convenciones |
| `architecture/module_map.md` | Mapa de modulos y dependencias |
| `architecture/infrastructure.md` | Infraestructura y configuracion |
| `tasks/` | Archivos de tareas (planificadas, en progreso, completadas) |
